# Thư mục Public

Thư mục này chứa các file tĩnh cho ứng dụng:

- `assets/` - Hình ảnh, sprite, icon
- `sounds/` - Âm thanh cho game
- `fonts/` - Font chữ tùy chỉnh

## Cấu trúc gợi ý:

```
public/
├── assets/
│   ├── background.png
│   ├── coin.png
│   ├── player.png
│   └── enemies/
├── sounds/
│   ├── click.mp3
│   ├── background.mp3
│   └── effects/
└── fonts/
    ├── game-font.woff2
    └── ui-font.woff2
```
